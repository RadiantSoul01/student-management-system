# student-management-system
Console-based Student Management System built with Java, applying Linked List (DSA) and Object-Oriented Programming



##  Overview

This is a console-based **Student Management System** built using Java.
The project demonstrates the use of **Data Structures (Linked List)** and **Object-Oriented Programming (OOP)** to manage student records dynamically.

---

##  Features

* ➕ Add student (Name, Registration Number, Marks)
* 🔍 Search student by Registration Number
* 📋 Display all students
* 📌 Dynamic data storage using Linked List

---

## 🧠 Concepts Used

* **Java OOP**

  * Classes & Objects
  * Encapsulation
  * Separation of concerns

* **Data Structures**

  * Singly Linked List
  * Traversal

* **Input Handling**

  * `Scanner` with `nextLine()` parsing (avoids buffer issues)
